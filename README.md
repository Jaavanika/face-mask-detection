# face-mask-detection
face mask detection using live systems

1. Installation
Clone the Repository

git clone https://github.com/jaavanika/face-mask-detection.git
cd face-mask-detection

2. Install Dependencies

pip install -r requirements.txt

3. Run the App

python app.py
